### Flusso del Traffico

1. **L'utente** installa l'app del provider VPN e si connette
2. L'app stabilisce un **tunnel crittografato** verso un server del provider
3. Tutto il traffico Internet dell'utente passa attraverso questo tunnel
4. Il traffico **esce su Internet** dal server VPN con l'IP del server
5. Le **risposte** dei siti web tornano al server VPN e vengono inoltrate all'utente attraverso il tunnel
