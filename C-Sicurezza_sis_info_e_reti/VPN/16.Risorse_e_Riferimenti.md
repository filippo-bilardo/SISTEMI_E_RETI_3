## 16.1 RFC (Request for Comments)

### IPsec Core Standards

| RFC | Titolo | Descrizione |
|-----|--------|-------------|
| **RFC 4301** | Security Architecture for IP | Architettura generale IPsec |
| **RFC 4302** | IP Authentication Header (AH) | Protocollo AH |
| **RFC 4303** | IP Encapsulating Security Payload (ESP) | Protocollo ESP |
| **RFC 5996** | IKEv2 Protocol | Internet Key Exchange v2 |
| **RFC 7296** | IKEv2 (updated) | Versione aggiornata IKEv2 |
| **RFC 4106** | AES-GCM for ESP | AES-GCM cipher suite |
| **RFC 3947** | NAT Traversal | NAT-T per IPsec |
| **RFC 3948** | UDP Encapsulation of IPsec | Incapsulamento UDP |

### TLS/SSL (per OpenVPN, SSL VPN)

| RFC | Titolo | Descrizione |
|-----|--------|-------------|
| **RFC 8446** | TLS 1.3 | Latest TLS version |
| **RFC 5246** | TLS 1.2 | TLS versione precedente |
| **RFC 6066** | TLS Extensions | SNI, OCSP Stapling |
| **RFC 7525** | TLS Best Practices | Raccomandazioni sicurezza |

### VPN Protocols

| **RFC 2637** | PPTP | Point-to-Point Tunneling Protocol |
| **RFC 2784** | GRE | Generic Routing Encapsulation |

## 16.14 Cheat Sheets

### OpenVPN

```bash
# Quick reference
openvpn --config <file>              # Start OpenVPN
openvpn --genkey secret <file>       # Generate static key
systemctl status openvpn@<name>      # Check service status
journalctl -u openvpn@<name> -f      # Live logs
```

### strongSwan/IPsec

```bash
ipsec status                         # Connection status
ipsec statusall                      # Detailed status
ipsec up <conn>                      # Start connection
ipsec down <conn>                    # Stop connection
ipsec reload                         # Reload config
ip xfrm state                        # Show SAs
ip xfrm policy                       # Show policies
```

### WireGuard

```bash
wg-quick up <interface>              # Start VPN
wg-quick down <interface>            # Stop VPN
wg show                              # Show status
wg show <interface>                  # Show specific interface
wg genkey                            # Generate private key
wg pubkey < privatekey               # Derive public key
```

### Troubleshooting

```bash
# Connectivity
ping -c 3 <ip>                       # Basic ping
traceroute <ip>                      # Trace route
mtr <ip>                             # Combined ping+trace

# Packet capture
tcpdump -i <interface> -n            # Capture packets
tcpdump -i any -n udp port 1194      # OpenVPN traffic
tcpdump -i any -n esp                # IPsec ESP

# Firewall
iptables -L -v -n                    # List rules
iptables -L FORWARD -v -n            # FORWARD chain
ufw status verbose                   # UFW status

# Network config
ip addr show                         # Show interfaces
ip route show                        # Show routing table
ss -tuln                             # Show listening ports
```
